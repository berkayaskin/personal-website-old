Campgrounds

Campgrounds is an application for a fictional start up that allows you to add campgrounds for other users to comment and rate, as view campgrounds added by other users.

Features
  • Adding campgrounds with image, description and price.
  • User Authentication
  • Comments
Technologies

HTML5, CSS3, Bootstrap, Javascript, Node.js, Express.js, MongoDB, Passport.js
